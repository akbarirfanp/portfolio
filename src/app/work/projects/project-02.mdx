## Overview

In this project, I worked as a Full Stack Developer to develop the company's product during my internship. Its a Ticketing System Service that centralizes customer service operations and integrates seamlessly with Genesys Cloud, a third-party customer experience platform. The system enables customer support agents to manage support tickets generated from live chat and call center interactions, providing a unified workflow for issue tracking and resolution. Beyond application development, I was also responsible for designing the system architecture, database structure, and service interactions to ensure scalability, maintainability, and smooth integration between internal services and external platforms.

## Key Features

- **Ticket Management System**: Developed a complete ticket lifecycle, including ticket creation, assignment, status tracking, prioritization, and resolution, allowing customer support teams to efficiently manage customer requests.
- **Genesys Cloud Integration**: Integrated the application with Genesys Cloud to synchronize customer interactions from live chat and call center services. Customer conversations handled through Genesys can automatically create or update tickets, enabling agents to manage communication and support cases from a centralized system.
- **REST API Development**: Designed and implemented RESTful APIs to facilitate communication between frontend, backend services, and the Genesys platform.

## Technologies Used

- **Nuxt 4**: Frontend framework.
- **Hypervel**: Backend framework for building the API Gateway services.
- **Goravel**: Backend framework for building the core services.
- **Docker**: For containerizing the application.
- **Helm**: For containerizing the application.
- **Genesys Cloud**: For integrating with the Genesys Cloud API.

## Challenges and Learnings

One of the main challenges was integrating the application with Genesys Cloud, which required understanding external APIs, authentication mechanisms, and event-driven communication between systems. Ensuring reliable synchronization between customer interactions and ticket records was essential to maintain data consistency.
## Outcome

This project strengthened my experience in full-stack development, microservices architecture, system design, and third-party API integration while working with modern technologies such as Nuxt, Hypervel, Golang, Docker, Helm, and Genesys Cloud.
